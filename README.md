# to-do-app
# 📝 Gerenciador de Tarefas (To-Do List)

Este é um projeto de **Lista de Tarefas** desenvolvido como parte dos meus estudos no primeiro semestre de programação. O objetivo principal é consolidar conhecimentos básicos de estrutura de páginas web (HTML) e estilização (CSS).

---

## 🚀 Sobre o Projeto

O projeto consiste em uma interface web funcional para desktop, onde o usuário pode organizar suas atividades diárias. O foco foi criar um layout limpo, intuitivo e de fácil manutenção.

### Funcionalidades Planejadas:
- **Campo de Entrada:** Espaço para descrição detalhada da tarefa.
- **Seletor de Prioridade:** Classificação visual entre tarefas Urgentes, Importantes ou Normais.
- **Lista Dinâmica:** Exibição organizada das tarefas cadastradas.
- **Marcação de Conclusão:** Checkbox para indicar tarefas finalizadas com efeito visual de "riscado".

---

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica de todo o conteúdo.
* **CSS3:** Design responsivo focado em Desktop, utilizando técnicas de *Flexbox* para alinhamento.

---

## 📂 Estrutura do Código

Para facilitar o entendimento inicial, o projeto foi estruturado da seguinte forma:

1.  **Reset CSS:** Garante que o site tenha a mesma aparência em diferentes navegadores (Chrome, Firefox, Edge).
2.  **Container Principal:** Centraliza o conteúdo no monitor, limitando a largura para uma leitura confortável.
3.  **Sistema de Cores de Prioridade:**
    * 🔴 **Urgente:** Borda esquerda vermelha.
    * 🟡 **Importante:** Borda esquerda amarela.
    * 🟢 **Normal:** Borda esquerda verde.

---

## 📖 Como Executar

Como este é um projeto baseado em tecnologias Front-End básicas, você não precisa de servidores ou instalações complexas:

1.  Faça o download ou copie o código do arquivo `index.html`.
2.  Abra o arquivo em qualquer navegador web (arrastando o arquivo para dentro do navegador ou clicando duas vezes).
